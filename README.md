**Risk Prediction for Visually Impaired People**
This project is developed to help visually impaired people by detecting obstacles and predicting risks using YOLOv8 object detection technology.
**Features**
- Real-time object detection
- Risk prediction system
- Smart assistance for visually impaired people
- Fast and accurate detection
**Technologies Used**
- Python
- YOLOv8
- OpenCV
- Machine Learning
